AniCameraX
Камера для Android, собранная с помощью ИИ. Основная цель — создать приложение с максимально гибкими настройками под конкретное железо, сохранив при этом визуальный стиль Pixel Camera (Material 3).
Текущие возможности:

    Движок: CameraX с поддержкой ZSL (Zero Shutter Lag) и HDR+ (через CameraX Extensions).
    Интерфейс: Полностью на Jetpack Compose. Динамические цвета (Monet), адаптивные иконки.
    Функционал:
        Ручная экспокоррекция и фокус по тапу.
        Зум-контроллер и быстрые переключатели (0.5x, 1x, 2x).
        Индикатор уровня наклона (акселерометр).
        Запись видео с индикацией таймера.
    Конфигурация: Система профилей (CameraProfile) для раздельной настройки передней и задней камер.

Разработчикам
